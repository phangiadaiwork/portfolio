Architecture Principles

🗃️

Database-per-Service

Each service owns its database. No cross-DB access allowed. Data shared only through events.

📡

Event-First

Asynchronous communication via RabbitMQ. Synchronous calls only when absolutely mandatory.

🎫

Token is Contract

JWT carries all auth context. No service-to-service auth calls needed. Zero-trust approach.

♻️

Eventual Consistency

Services are eventually consistent. Soft delete with deletedAt pattern and partial unique indexes.
